# Basic Firewall Configuration with UFW

## Objective
Configure a basic firewall using UFW on Kali Linux.

## Tools Used
- Kali Linux
- UFW
- Bash Shell

## Firewall Rules

- Allow SSH (Port 22)
- Deny HTTP (Port 80)
- Allow HTTPS (Port 443)
- Deny FTP (Port 21)

## Verification

Firewall rules were verified using:

sudo ufw status verbose

## Conclusion

The firewall was successfully configured using UFW.
